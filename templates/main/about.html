Справка

Веб-сервис предназначен для разработки шаблонов документов и использования их для создания документов Microsoft Word. Веб-сервис может брать данные из базы данных организации, для этого необходимо системному администратору предварительно настроить подключение к базе данных.

Документ состоит из элементов нескольких типов:

  • Текст - представляет из себя элемент, содержащий в себе текст и определённый стиль оформления. Каждый текст может быть оформлен в своём стиле и является обособленным от других.
  • Параграф - представляет из себя элемент, содержащий в себе множество других элементов, в основном текстов.
  • Переменная - является текстом, который берёт в себя данные из базы данных.

Создание элементов

Чтобы создать параграф нажмите на кнопку с иконкой

Чтобы создать текст нажмите на кнопку с иконкой

Чтобы создать переменную вам нужно перейти в специальное меню работы с переменными. В этом меню вы можете заполнить переменные организации для поиска или создать свою переменную для данного документа. Чтобы отобразить переменную в документе вам нужно создать текст, после нажать на поле ввода выше документа и выбрать нужную переменную из списка либо найти её при помощи поиска.

Горячие клавиши при работе с документом

При наборе текста:

Добавить новый абзац: Enter

Добавить новый текст: Shift + Пробел

Переключаться между текстовыми элементами: Ctrl + Стрелочки

Удаление элемента: Delete
